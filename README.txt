INTRUCTIONS:
1. Start the program with python3 musicApp.py
2. A new window will pop up with the application
3. Create an account as an Attendee or an Organizer 
4. Sign in with your previous login information 
5. As an Attendee, you can log in and explore events, artists, venues, etc
6. As an Organizer, you can log in and view venue capacity and schedule your artists/events

Note: The csv files for tickets and events will appear in your current directory.
A file path will be displayed. Go to your files to view and open the csv file
